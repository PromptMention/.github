# PromptMention

PromptMention builds tools that help brands understand and improve how they show up across AI-powered discovery surfaces.

Website: [promptmention.com](https://promptmention.com)

## What we build

- **AI visibility tooling** for brands, publishers, and commerce teams
- **Open-source infrastructure** for agent-ready websites and machine-readable web standards
- **Practical utilities** around discoverability, content accessibility, and agent interaction patterns

## Featured project

- [**isitagentready**](https://github.com/PromptMention/isitagentready): open-source scanner for checking whether a website supports emerging AI agent standards such as `robots.txt`, markdown negotiation, API catalogs, MCP server cards, and agent skills discovery

## Focus areas

- AI visibility
- agent-ready websites
- markdown for agents
- MCP and agent discovery
- search and content infrastructure
